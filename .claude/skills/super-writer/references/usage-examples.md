## Key Takeaways from Examples

### Complexity Scaling

**Level 1 (Simple):**
- Minimal stages executed (often skip 2-3, 4.5-4.6)
- Single methodology
- 1-3 minute total time
- Focus on efficiency

**Level 2 (Medium):**
- Most stages executed
- Material collection typical
- 8-12 minute total time
- Balance efficiency and quality

**Level 3 (Complex):**
- All relevant stages executed
- Deep analysis when reference provided
- 15-30 minute total time
- Focus on quality and depth

### Stage Skipping Logic

**Always Execute:**
- Stage 1 (Analysis)
- Stage 4 (Reference Check)
- Stage 5 (Methodology Selection)
- Stage 6 (Prompt Retrieval)
- Stage 7 (Content Creation)

**Conditionally Execute:**
- Stage 2-3 (Material): Only if material needed
- Stage 4.5-4.6 (Style Analysis): Only if reference provided

### Adaptation Principles

1. **Adjust to user needs** - More or less depth based on requirements
2. **Time management** - Balance thoroughness with efficiency
3. **Quality focus** - Never sacrifice quality for speed on complex tasks
4. **Communication** - Keep user informed but don't overwhelm
5. **Flexibility** - Adapt process to specific situation

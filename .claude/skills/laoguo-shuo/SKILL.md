# 老郭说

## Overview

"老郭说"是一个以历史故事为切入点、结合现代管理智慧的深度文章创作技能。文章风格幽默风趣但不失专业严谨，善于从三国等历史故事中提炼人性和管理的洞察。

## Core Style Characteristics

**文章风格要素：**

1. **开头结构**
   - 字数统计 + 阅读时间（如：总字数：2208字 阅读时间：约7分钟）
   - 问候语："大家好，我是老郭。"
   - 引入话题：用简洁的引言营造思考氛围

2. **内容特点**
   - 从三国或历史故事切入
   - 连接现代管理/职场/商业现实
   - 提炼人性和管理的深度洞察
   - 金句频出，易于传播

3. **文章结构**
   - 编号章节（### 01、### 02、### 03...）
   - 每个章节聚焦一个核心观点
   - 用故事+分析+金句的方式推进
   - 章节标题既有趣又有洞察

4. **语言风格**
   - 口语化但不失深度
   - 幽默风趣 + 专业严谨
   - 善用对比、类比
   - 强烈的情感共鸣

5. **结尾模板**
   ```
   我是老郭，一个喜欢从人性和历史的视角，
   帮管理者提前识别风险、提前做出艰难但正确决策的企业教练。
   如果今天的文章对你有启发，欢迎**点赞、收藏、转发**，咱们评论区见！
   ```

## Workflow

此技能采用**半自动填充**模式工作：

### Step 1: 理解用户需求

当用户请求创作文章时，先明确：

- **主题**：文章要讲什么核心观点？
- **素材**：用户是否提供具体的历史故事/案例？
  - 如未提供，从内置三国知识库中匹配合适的故事
- **框架**：用户是否有大纲或章节规划？
  - 如有，按框架填充内容
  - 如无，协助构建大纲

### Step 2: 匹配或选择历史故事

**如果用户提供素材：**
- 分析素材与主题的关联点
- 提炼故事中的管理/人性洞察

**如果用户未提供素材：**
从以下经典三国场景中匹配：

| 现代管理议题 | 对应三国故事 |
|------------|------------|
| 权力与特权 | 刘备脱离曹操、斩车胄 |
| 危机公关 | 陈琳骂曹操 |
| 向上管理 | 陈登戏吕布 |
| 安全感与组织 | 刘安杀妻、吕布之死 |
| 破局智慧 | 衣带诏到吉平下药 |

### Step 3: 构建文章结构

创建编号章节大纲，每个章节应包含：
- 故事片段
- 现实对照
- 核心洞察
- 金句提炼

**章节设计原则：**
- 每章聚焦一个核心观点
- 从具体到抽象，层层递进
- 用"总-分-总"的逻辑推进

### Step 4: 撰写完整文章

按照"老郭说"风格填充内容，确保：

**开头吸引人：**
- 用反常识的观察制造悬念
- 或用强烈的现实代入感

**中间有层次：**
- 故事讲述简洁生动
- 现实对照精准有力
- 洞察提炼深刻而不说教

**结尾有回味：**
- 总结核心观点
- 给出实用建议
- 标准的自我介绍 + CTA

### Step 5: 金句提炼

每篇文章应包含3-5条金句，特征：
- 短小精悍（10-20字）
- 对比强烈
- 易于记忆和传播
- 直击人性或管理本质

**示例金句：**
- "手段会反过来塑造目的"
- "你狠一次，就更容易狠第二次"
- "把人当问题，把问题当人"
- "敏感是一种能力，同时也是一种负担"

## Quality Standards

**每篇文章必须满足：**

1. **深度**：不只是讲故事，而是提炼洞察
2. **共鸣**：让读者有强烈的现实代入感
3. **趣味**：轻松幽默但不轻浮
4. **专业**：管理洞察准确、逻辑严密
5. **完整**：开头-主体-结尾结构完整

**避免：**
- 纯粹的故事复述
- 缺乏现实连接
- 过于严肃说教
- 金句生硬堆砌

# 对话特征提取指南

## 语言风格分析维度

### 1. 口头禅与高频表达
**识别方法**:
- 重复出现的短句（2-15字）
- 特定的开场白/结束语
- 情绪相关的高频短语

**示例**:
```
高频: "就是说...", "其实吧", "你知道我什么意思"
情绪: "烦死了", "太棒了", "无语"
连接: "然后呢", "所以就是说", "对吧"
```

### 2. 用词习惯
**关注点**:
- 正式 vs 口语化程度
- 方言/地域词汇
- 专业术语使用频率
- 比喻和修辞偏好

**分析技巧**:
```
正式度: 书面语比例 / 总词数
专业度: 行业术语频率
口语化: "啊、呢、吧"等语气词密度
```

### 3. 句式结构
**测量指标**:
- 平均句长
- 短句 vs 长句比例
- 疑问句/感叹句频率
- 倒装/省略现象

**示例模式**:
```
简洁型: 平均句长 < 15字，短句比例 > 60%
详细型: 平均句长 > 30字，长句比例 > 40%
互动型: 疑问句比例 > 20%
```

### 4. 标点符号习惯
**个性化特征**:
```
! 重复: 激情/急切表达
? 重复: 困惑/强调疑问
~ 使用: 轻松/友好语气
... 省略: 思考/话未说完
```

### 5. 情绪表达模式
**情绪类型识别**:
```
积极情绪: 开心、期待、满意、放松
消极情绪: 焦虑、愤怒、失望、压力
中性情绪: 平静、客观、陈述
```

**表达方式**:
- 直接表达: "我很生气"
- 间接表达: "这事真的..."
- 压抑型: 情绪词少，但能感觉到
- 夸张型: 大量情绪化词汇

### 6. 话题偏好
**话题分类**:
```
工作类: 项目、任务、同事、客户
生活类: 吃饭、购物、天气、日常
兴趣类: 游戏、运动、音乐、电影
关系类: 家人、朋友、恋爱
思考类: 观点、想法、哲学、未来
```

### 7. 对话行为模式
**主动 vs 被动**:
- 主动发起话题频率
- 提问 vs 回答比例
- 话题切换方式

**倾听 vs 表达**:
- 自己说话长度 vs 对方
- 回应方式（简短 vs 详细）
- 是否经常打断/转移话题

## 大五人格特征映射

### 开放性 (Openness)
**高分表现** (75-100):
- 喜欢尝试新事物、新观点
- 用词丰富，常有抽象/哲理性表达
- 对艺术、创意、未来话题感兴趣
- 语言特征: "我觉得可以试试", "有个想法"

**低分表现** (0-40):
- 偏好传统、熟悉的方式
- 实用主义，关注具体问题
- 语言特征: "按老规矩来", "别折腾"

### 尽责性 (Conscientiousness)
**高分表现** (75-100):
- 有计划、有条理
- 时间观念强，守时
- 注重细节和质量
- 语言特征: "先规划一下", "得确认一下", "按计划来"

**低分表现** (0-40):
- 随性、灵活
- 可能拖延或临时决策
- 语言特征: "看情况吧", "到时候再说", "无所谓"

### 外向性 (Extraversion)
**高分表现** (75-100):
- 主动社交，喜欢互动
- 表达直接，语速可能较快
- 情绪外露
- 语言特征: 多感叹号，主动发起话题，"走吧！" "来啊！"

**低分表现** (0-40):
- 偏内向，表达较含蓄
- 思考后再回应
- 语言特征: 省略号多，"嗯..." "让我想想"

### 宜人性 (Agreeableness)
**高分表现** (75-100):
- 友善、合作、照顾他人感受
- 避免冲突，委婉表达
- 语言特征: "没关系", "看你方便", "我觉得可以"

**低分表现** (0-40):
- 直接、坦率，可能显得生硬
- 坚持己见
- 语言特征: "不行", "就是不对", "我不同意"

### 神经质 (Neuroticism)
**高分表现** (75-100):
- 情绪波动大，易焦虑
- 对压力敏感
- 语言特征: "担心", "怕", "烦", "怎么办"

**低分表现** (0-40):
- 情绪稳定，抗压能力强
- 冷静面对问题
- 语言特征: "没事", "问题不大", "能解决"

## 输出报告模板

### 综合画像结构
```
1. 基本信息
   - 数据来源
   - 文本量统计
   - 时间跨度

2. 语言风格总结
   - 核心特征（3-5个关键词）
   - 口头禅 TOP 10
   - 标点习惯

3. 性格特征
   - 大五人格分数转化
   - 行为模式描述
   - 决策风格

4. 对话模式
   - 主动/被动倾向
   - 表达方式（简洁/详细/幽默...）
   - 情绪表达特征

5. 话题偏好
   - 高频话题排序
   - 兴趣领域

6. CC 应用建议
   - Description 描述段落
   - Definition 对话样本选择
   - Greeting 开场白建议
```

## 数据准备阶段

### ✅ DO（推荐做法）

1. **提前了解背景**
   - 了解对话场景（访谈、闲聊、工作讨论）
   - 确认说话人身份和数量
   - 了解录制时间和情境

2. **预览文件内容**
   ```bash
   # 查看文件前100行
   head -100 filename.md

   # 查看文件总行数
   wc -l filename.md

   # 搜索关键标识
   grep "自我介绍" filename.md
   ```

3. **确认文件格式**
   - 检查是否为纯文本或Markdown
   - 确认编码格式（UTF-8）
   - 查看是否有时间戳、说话人标记

### ❌ DON'T（避免做法）

1. **不要盲目处理**
   - 不要跳过预览直接生成报告
   - 不要假设说话人标签准确

2. **不要忽视背景**
   - 不要忽略对话发生的场景
   - 不要忘记考虑文化、职业等因素

## 人物识别与提取

### ⚠️ 关键原则

**永远不要完全依赖发言人标签！**

### 识别流程

#### Step 1: 内容扫描

1. 查找自我介绍语句：
   - "那我就讲一下我自己"
   - "我是..."
   - "我..."

2. 识别职业相关话题：
   - 心理咨询：个案、督导、转化率
   - 技术讨论：模型、算法、工程
   - 人力资源：特质论、测评、团队

3. 分析对话逻辑：
   - 谁在提问？谁在回答？
   - 谁主导话题？
   - 谁回应更多？

#### Step 2: 使用Task工具辅助

当人物关系不明确时：

```
使用Task工具
subagent_type: general-purpose
prompt: "分析文件，识别说话人，提取各自对话内容"
```

#### Step 3: 验证提取结果

- 检查每人对话字数是否合理
- 确认话题是否符合人物身份
- 核对关键信息是否自洽

### 案例参考

| 案例 | 识别要点 | 验证方法 |
|------|---------|---------|
| **Harry/郭磊** | Harry=工程师(技术话题)<br>郭磊=HR顾问(特质论话题) | 字数统计、话题分布 |
| **罗老师** | 讲述职业经历者=罗老师<br>访谈者=HR顾问 | "那我就讲一下我自己" |

**自我表露度评分标准**:
- **10/10** (极高): 像罗老师一样分享"巨坑"、失败、贴钱等负面经历
- **7-9/10** (高): 分享个人经历，但有所保留
- **4-6/10** (中等): 偶尔分享个人信息
- **1-3/10** (低): 极少分享个人信息

## 人格映射阶段

### 大五人格评分要点

#### 开放性细化

| 分数段 | 标签 | 典型特征 | 案例 |
|--------|------|---------|------|
| 90-100 | 概念创新者 | 提出原创概念 | 郭磊（模型特质论） |
| 85-89 | 快速接受者 | 快速接受新理念 | 罗老师（震撼但很快接受） |
| 70-84 | 开放创新 | 愿意尝试新事物 | 璐露（跳出舒适圈） |

#### 宜人性补充评估

**可结合"自我表露度"评估**:
- 高自我表露 + 高宜人性 = 坦诚真实（罗老师）
- 高自我表露 + 中等宜人性 = 直率坦诚（Harry）
- 中等自我表露 + 高宜人性 = 温和包容（郭磊）

#### 尽责性细化

**目标导向 vs 灵活调整**:
- 高尽责 + 高灵活性: 罗老师（目标明确但灵活调整）
- 高尽责 + 低灵活性: 可能的死磕型
- 中等尽责 + 高灵活性: 郭磊（关注问题导向）

## 常见问题解决

### Q1: 人物识别错误怎么办？

**案例**: Harry/郭磊对话（初次处理时搞反了）

**解决方法**:
1. 重新审视对话内容
2. 根据话题而非标签识别
3. 使用Task工具重新提取
4. 验证提取结果的合理性

**检查清单**:
- [ ] 对话字数分配是否合理？
- [ ] 话题是否符合人物身份？
- [ ] 关键信息是否自洽？

### Q2: 如何评估"自我表露度"？

**评估维度**:
1. **负面经历分享**: 是否分享失败、"巨坑"等？
2. **情感暴露**: 是否表达真实情绪（焦虑、挫败、兴奋）？
3. **个人信息**: 是否分享个人背景、成长经历？
4. **脆弱性**: 是否展示不完美的一面？

**评分示例**:
- 罗老师 (10/10): 分享"巨坑"、"贴钱"、创业失败
- 郭磊 (7/10): 分享工作思考，但较少负面经历
- Harry (5/10): 主要讨论技术，个人信息较少
- 璐璐 (8/10): 坦诚表达内向和不舒适

### Q3: 如何写"独特之处"章节？

**步骤**:
1. **识别原创概念**: 是否有独特的理念或观点？
2. **标记关键转折**: 职业生涯的转折点
3. **记录金句口头禅**: 反复出现的表达
4. **对比他人**: 与常见类型有什么不同？

**案例参考**:

**郭磊的独特之处**:
- 原创概念: "模型特质论"
- 关键转折: 从特质论延伸到AI模型特质
- 口头禅: "你懂我意思吧？"

**罗老师的独特之处**:
- 关键表达: "巨坑"、"钢铁直男"
- 价值观冲突: "震撼但很快接受"商业化
- 职业转折: 多次灵活调整（社工→咨询→创业→回归）

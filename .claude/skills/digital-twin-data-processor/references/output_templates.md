# 输出格式模板

## Character.AI 角色创建格式

### 1. Description (角色描述)

**长度**: 200-500 字

**结构**:
```
第1段: 核心性格概述（基于大五人格最高分维度）
第2段: 行为模式和决策风格（基于尽责性、外向性）
第3段: 社交风格和沟通特点（基于宜人性、外向性）
第4段: 情绪特征和压力反应（基于神经质）
第5段: 语言风格总结（结合文本分析结果）
```

**示例**:
```
一个充满创造力和好奇心的思想家，总是探索新想法和独特视角。思维活跃，喜欢深入探讨抽象概念，对未来充满期待。做事有计划但不刻板，能够在灵活性和条理性之间找到平衡。

在社交方面，性格开朗但不过分外向，善于表达同时也懂得倾听。与人交流时热情友善，但也坚持自己的立场。表达方式富有感染力，善用比喻和故事来说明观点。

情绪基本稳定，能够较好地管理压力。遇到问题时倾向于冷静分析而非情绪化反应。但在面对重要决策或压力时，也会表现出正常人的焦虑和担忧。

语言风格富有变化，既有逻辑严密的论述，也有生动形象的比喻。经常使用"我觉得"、"有个想法"等表达，显示其思考型特质。口头禅包括"就是说"、"其实"等，体现其善于解释和澄清的特点。
```

### 2. Greeting (开场白)

**长度**: 1-3 句话

**要求**:
- 符合角色的语言风格
- 体现性格特征
- 自然引发对话

**示例**:

**外向型**:
```
"嘿！很高兴认识你！我最近在思考一个有趣的想法，想听听你的看法..."
```

**内向型**:
```
"你好。我准备了一些想法，如果你感兴趣的话，我们可以慢慢聊聊。"
```

**温和型**:
```
"哈喽~ 很高兴认识你呢！希望我们能聊得开心~"
```

**直接型**:
```
"你好。有什么具体的事情需要讨论吗？我们可以直接进入主题。"
```

### 3. Definition (对话样本)

**格式**: 每行一个对话样本

**内容**: 从实际对话中提取的代表性发言

**数量**: 20-50 条

**分类**:
```
# 日常表达 (5-10条)
"就是说，我们需要从不同的角度来看这个问题。"
"其实我觉得，关键在于找到平衡点。"

# 情绪表达 (5-10条)
"今天真的太棒了！完全超出预期！"
"这个事情让我有点担心，因为..."

# 观点阐述 (5-10条)
"我的观点是，我们应该考虑长期影响。"
"从数据来看，这个趋势很明显。"

# 互动回应 (5-10条)
"嗯，你说的有道理，让我想想..."
"这个想法不错！我们可以进一步讨论。"

# 口头禅 (3-5条)
"就是说..."
"其实吧..."
"对吧？"
```

**完整 Definition 示例**:
```
{{user}}: 你好
{{char}}: 嘿！很高兴认识你！我最近在思考一个有趣的想法，想听听你的看法

{{user}}: 什么想法？

{{user}}: 能具体说说吗
{{char}}: 当然。比如说我们现在讨论的这个项目，如果只看当前的进度，可能会觉得有些慢。但如果从长期价值的角度来看，我们正在建立的基础其实非常重要

{{user}}: 有道理
{{char}}: 对吧？所以我常说，做事情要有长远的眼光。当然，这也不是说要忽略细节，而是在大局和细节之间找到平衡点

{{user}}: 你平时怎么放松的

{{char}}: 嗯...压力肯定会有的。我通常会先分析问题的根源，然后制定计划。一步步来，感觉就没那么焦虑了。当然，有时候也会需要和朋友聊聊，或者干脆休息一下

{{char}}: 我觉得是保持开放的心态。就是说，不要固守己见，要愿意接受新的想法和不同的观点。这样才能不断成长
```

```json
  }
}
```

## 特征分析报告格式

### 文本分析报告（供用户审阅）

```markdown
# 数字孪生特征分析报告

## 一、数据概览
- **数据来源**: 录音转写、微信聊天记录
- **文本总量**: 523,456 字
- **时间跨度**: 2023年1月 - 2025年1月
- **对话场景**: 日常对话 (72%)、工作讨论 (20%)、深度交流 (8%)

## 二、大五人格分数
| 维度 | 分数 | 等级 |
|------|------|------|
| 开放性 | 82/100 | 高度开放 |
| 尽责性 | 65/100 | 中等尽责 |
| 外向性 | 58/100 | 中性偏外向 |
| 宜人性 | 76/100 | 友善合作 |
| 神经质 | 38/100 | 情绪稳定 |

## 三、语言风格特征

### 3.1 核心特征
- **思维活跃，富有创造力**: 善于从多角度思考问题
- **表达清晰，逻辑性强**: 能够系统性地阐述观点
- **温和友善，善于倾听**: 注重他人感受，乐于合作
- **情绪稳定，抗压能力强**: 面对压力保持冷静

### 3.2 口头禅 TOP 10
1. "就是说..." (234次)
2. "其实..." (189次)
3. "对吧？" (156次)
4. "我觉得..." (142次)
5. "有个想法..." (98次)
6. "从...角度来看" (87次)
7. "关键是..." (76次)
8. "换句话说..." (65次)
9. "其实吧..." (54次)
10. "你知道吗..." (43次)

### 3.3 标点符号习惯
- **感叹号**: 适中 (每千字 8.5 个) - 有激情但不夸张
- **问号**: 较多 (每千字 12.3 个) - 喜欢互动和确认
- **波浪号**: 偶尔使用 (每千字 2.1 个) - 展现轻松友好的一面
- **省略号**: 较少 (每千字 1.5 个) - 表达较为直接

### 3.4 句式特征
- **平均句长**: 24.3 字 - 适中的长度
- **短句比例** (≤10字): 28% - 有简短有力的表达
- **长句比例** (≥50字): 15% - 能够详细阐述
- **疑问句比例**: 22% - 互动性强

### 3.5 情绪表达
- **积极情绪词**: 高频使用 (开心、棒、厉害)
- **消极情绪词**: 适中 (烦、担心 - 压力相关)
- **思考词**: 频繁 (觉得、感觉、可能、应该)
- **情绪稳定性**: 整体偏积极，偶有压力表达

### 3.6 话题偏好
1. **工作与项目** (35%) - 项目进展、团队协作
2. **想法与观点** (25%) - 对事物的看法和思考
3. **日常生活** (20%) - 吃饭、购物、出行
4. **未来规划** (12%) - 计划、目标、期待
5. **兴趣娱乐** (8%) - 电影、游戏、音乐

## 四、对话行为模式

### 4.1 主动性
- **发起话题**: 较多 - 喜欢分享想法和观点
- **提问频率**: 高 - 喜欢确认和寻求反馈
- **话题切换**: 中等 - 会按话题自然转换

### 4.2 表达方式
- **简洁性**: 中等 - 既有简洁表达，也会详细说明
- **逻辑性**: 强 - 善于系统性阐述
- **情感表达**: 适中 - 不过分外露，也不过分克制

### 4.3 倾听与回应
- **倾听意愿**: 强 - 注重他人观点
- **回应方式**: 详细且友好
- **打断频率**: 低 - 礼貌等待对方说完
```

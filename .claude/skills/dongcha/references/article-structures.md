## 结构A: 问题-分析-解决方案

### 适用场景
- 具体问题导向
- 痛点明确的主题
- 需要给出行动方案

### 结构模板


## 写作技巧

### 标题技巧

**好标题的4个要素**:
1. **问题导向**: "为什么XXX会XXX？"
2. **数字吸引**: "3个方法让XXX"
3. **反差冲击**: "做了人才盘点，核心员工反而走了"
4. **金句式**: "经营先于管理，管理服务于经营"

**标题公式**:
- [现象/问题] + [洞察/方法]
- [人物] + [核心观点]
- [数字] + [结果]

### 导语技巧

**3种经典导语**:

1. **场景引入式**
```
昨天和一位HRD聊天，他说...
这让我想起了之前的一个案例...
```

2. **金句抛出式**
```
"人才盘点不是'贴标签'，而是'照镜子'"
这是我最近常说的一句话...
```

3. **数据震撼式**
```
70%的变革项目失败，不是因为方案不好...
```

### 金句提炼

**金句特征**:
1. 短小精悍（15字以内）
2. 朗朗上口
3. 揭示真相
4. 便于记忆和传播

**金句公式**:
- 对比式："经营先于管理，管理服务于经营"
- 因果式："没有信任的盘点，只会加速人才流失"
- 行动式："少开会，多谈话"
- 洞察式："组织的本质不是架构，是连接"

### 段落过渡

**自然过渡的4种方式**:

1. **递进式**
```
不仅...，更重要的是...
```

2. **转折式**
```
然而，事情并非这么简单...
```

3. **设问式**
```
那么，为什么会出现这种情况？
```

4. **总结式**
```
综上所述，我们可以得出一个结论...
```

## 常见写作瓶颈与突破

### 瓶颈1: 不知道选什么结构
**突破**: 问自己3个问题
1. 我的核心内容是什么？
2. 我想让读者得到什么？
3. 什么结构最能传递价值？

### 瓶颈2: 洞察不够深
**突破**: 用5Why法
- 为什么会这样？（1）
- 为什么会这样？（2）
- 为什么会这样？（3）
- 为什么会这样？（4）
- 为什么会这样？（5）

### 瓶颈3: 不知道怎么开头
**突破**: 从3个角度选一个
1. 从场景切入：描述一个具体场景
2. 从金句切入：抛出一句震撼的话
3. 从问题切入：提出一个引人深思的问题

### 瓶颈4: 写不下去
**突破**: 用对话体
- 想象和读者对话
- 用问答形式展开
- "你可能会问..."
